# Poetry project boiler plate

A repository template using Poetry, Blake, and pre-commit-hooks

## Project structure

```bash
.
.
├── .DS_Store
├── .flake8
├── .gitattributes
├── .gitignore
├── .pre-commit-config.yaml
├── README.md
├── main.py
├── poetry.lock
├── poetry.toml
├── pyproject.toml
└── settings.json
```
